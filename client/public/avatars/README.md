# Avatares de jugadores

Esta carpeta queda lista para cargar fotos reales de los jugadores sin
comitearlas al repo.

Convención:

- Un archivo por jugador.
- Nombre: `<playerId>.webp`, `<playerId>.jpg` o `<playerId>.png`.
- IDs disponibles: `javi`, `nico`, `frang`, `lucho`, `tincho`, `mati`, `seba`.

En dev y en producción las fotos se sirven como `/avatars/<playerId>.<ext>`.
Por ejemplo: `/avatars/javi.jpg`.
